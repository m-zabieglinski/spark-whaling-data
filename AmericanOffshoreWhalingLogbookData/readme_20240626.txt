Format

This download package includes the database in a single file as tab-delimited text. The file has column names in the first row and 467,053 data rows. It is UTF-8 encoded with Unix line endings between records and no line breaks embedded within any data. 


Additions & corrections

Users who have facts to add or who discover errors in the database are encouraged to submit them. We plan to update the online database annually. https://whalinghistory.org/participate/corrections/


Columns

0	VoyageID
1	Lat
2	Lon
3	Day
4	Month
5	Year
6	Encounter
7	Species
8	NTried
9	PlaceName
10	Source
11	sequence

Column Definitions & Code Tables are available on the Whaling History website.
